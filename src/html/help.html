Google Grades Filter Help

If you're seeing this page, you've already cleared the first hurdle to getting help - well done! But admittedly, it is still pretty rudimentary.

In short, this extension allows you to filter the entries on the screen when you're on the Grades tab in your Google Classroom... enter the search text in the input field... that's it; that's all.
